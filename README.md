Calculadora em Python (Tkinter) — Versão Refatorada

Sobre o projeto
Esta é uma versão refatorada e aprimorada de uma calculadora desenvolvida em Python com Tkinter.
O projeto surgiu a partir de uma implementação inicial baseada em tutorial e foi reorganizado
com foco em clareza, organização e boas práticas de programação.

O objetivo principal desta versão é simular um processo real de evolução de código,
partindo de uma solução funcional, porém frágil, para uma solução mais estruturada e previsível.


Evolução do projeto
Este projeto é a continuação direta da versão inicial:
 *([link para o projeto inicial](https://github.com/NINGUEM-exe/calculadora-tkinter-inicial))*

A partir dela, foram aplicadas melhorias estruturais e conceituais.


Objetivos desta versão
- Melhorar a organização do código
- Reduzir dependência de variáveis globais
- Separar responsabilidades (interface x lógica)
- Tornar o comportamento da calculadora previsível
- Facilitar manutenção e futuras melhorias
- Aplicar boas práticas aprendidas durante os estudos


Principais melhorias aplicadas
- Organização do código em funções mais claras
- Melhor controle do estado da aplicação
- Tratamento mais seguro da lógica de cálculo
- Correção de comportamentos inconsistentes dos botões
- Código mais legível e fácil de entender
- Estrutura pensada para evolução futura


Tecnologias utilizadas
- Python 3
- Tkinter
- Git
- GitHub
